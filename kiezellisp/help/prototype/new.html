Name

prototype:new

Usage

builtin-constructor

Syntax

(prototype:new &rest args)

Description

Creates a new prototype object. When the number of arguments is odd, the first argument must be either a type specifier or a list of type specifiers. A type specifier is either a prototype object or the name of a type created by deftype, defclass, defstruct or import. The other arguments are name/value pairs used to initialize the new object. Names can be given as strings, symbols and keywords.

user > (let a (new :color "black"
                   :model "0"))
it: a
user > (new a
            :model "12")
it: #s( :color "black"
        :model "12" )
