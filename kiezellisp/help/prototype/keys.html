Name

prototype:keys

Usage

builtin-function

Syntax

(prototype:keys object)

Description

Returns a list of strings.

user > (let z (new "name" "Piet"
                   :city "Amsterdam"))
it: z
user > (prototype:keys z)
it: ("name" "city")
