Name

prototype:get-type-specifier

Usage

builtin-function

Syntax

(prototype:get-type-specifier obj)

Description

Returns a type name or a list of the type names of the prototype object obj.

user > (deftype a1 (new :city "leiden"))
it: #s(a1 :city "leiden" )
user > (deftype a2 (new :country "netherlands"))
it: #s(a2 :country "netherlands" )
user > (def b (new 'a1))
it: b
user > (prototype:get-type-specifier b)
it: a1
user > (def c (new '(a1 a2)
                   :name "mies"))
it: c
user > c
it: #s((a1 a2) :city "leiden"
               :country "netherlands"
               :name "mies" )
user > (prototype:get-type-specifier c)
it: (a1 a2)
