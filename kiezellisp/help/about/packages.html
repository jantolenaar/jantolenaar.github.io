Name

about:packages

Description

Kiezellisp packages are almost like Common Lisp packages. The import of a .NET type creates a package that stores the members of the type.

Package Alias

The scope of a package alias is the current package.

(in-package "aaaaaaaaaa")
(use-package-alias "cccccc" "c")
    cccccc:name -> ok
    c:name      -> ok
(in-package "bbbbbbbbbb")
    cccccc:name -> ok
    c:name      -> error
