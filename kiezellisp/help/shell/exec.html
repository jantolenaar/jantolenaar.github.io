Name

shell:exec

Usage

builtin-function

Syntax

(shell:exec input prog &rest args)

Description

Executes the program prog with arguments args. If the string input is not null, it is used as standard input. Returns the standard output as a string.

user > (shell:exec null "uname" "-a")
it: "Linux debian 4.9.0-8-amd64 #1 SMP Debian 4.9.144-3.1 (2019-02-19) x86_64 GNU/Linux\n"
