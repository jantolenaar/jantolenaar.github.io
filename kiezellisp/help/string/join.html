Name

string:join

Usage

builtin-function

Syntax

(string:join separator values)
(string:join separator value start-index count)
(string:join separator seq)

Description

The third form is a Kiezellisp extension.

user > (.join ", " '(aap noot mies))
it: "aap, noot, mies"
