Name

string:trim-start

Usage

builtin-function

Syntax

(string:trim-start str)
(string:trim-start str &rest trim-chars)
(string:trim-start str trim-chars-str)

Description

The third form is a Kiezellisp extension.

user > (.trim-start "abcdefefefef" "fe")
it: "abcdefefefef"
