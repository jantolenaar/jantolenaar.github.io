Name

string:indent-with-line-numbers

Usage

builtin-function

Syntax

(string:indent-with-line-numbers text line-number width separator)

Description

Adds a line number and prefix string to each line in text.

user > (print-line (.indent-with-line-numbers "aaaa\nbbbb" 12 4 ": "))
