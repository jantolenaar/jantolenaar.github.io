Name

string:try-parse-date

Usage

builtin-function

Syntax

(string:try-parse-date str &key format culture)

Description

Converts a string to a DateTime.

make-html > (.try-parse-date "20130104" :format "yyyyMMdd")
it: "2013-04-01"
make-html > (.try-parse-date "20139904" :format "yyyyMMdd")
it: null
