Name

string:regex-match

Usage

builtin-function

Syntax

(string:regex-match str pattern)

Description

Compares str to the string or Regex pattern. Returns null if no match. Returns the first match otherwise.

user > (.regex-match "hello, world" "(.)rl")
it: ("orl" "o")
