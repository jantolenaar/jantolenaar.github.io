Name

string:regex-split

Usage

builtin-function

Syntax

(string:regex-split str pattern)

Like string:split but with separators defined by the regular expression pattern.

user > (.regex-split "een1twee22drie333vier" #/\d/)
it: ["een" "twee" "" "drie" "" "" "vier"]
