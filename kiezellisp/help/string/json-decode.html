Name

string:json-decode

Usage

builtin-function

Syntax

(string:json-decode str)

Description

Decodes a JSON string.

make-html > (string:json-encode (new :name "mies"))
it: "{ \"name\": \"mies\" }"
make-html > (.json-decode it)
it: #s( :name "mies" )
