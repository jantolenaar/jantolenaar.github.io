Name

math:truncate

Usage

builtin-function

Syntax

(math:truncate number)

Description

Returns the nearest integer in the direction of zero.

make-html > (math:truncate 6.99)
it: 6
make-html > (math:truncate 14/3)
it: 4
make-html > (math:truncate -14.3)
it: -14
