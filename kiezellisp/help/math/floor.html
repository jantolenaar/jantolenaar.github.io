Name

math:floor

Usage

builtin-function

Syntax

(math:floor number &optional (denominator 1))

Description

Returns a multiple value with the quotient and remainder of the division. The quotient is rounded to the nearest integer in the direction of negative infinity.

user > (math:floor 6)
it: 6
    0
user > (math:floor 14 3)
it: 4
    2
user > (math:floor 14/2 3/2)
it: 4
    1
user > (math:floor 14.0 3.0)
it: 4
    2.0
user > (math:floor 14 -3)
it: -5
    -1
user > (math:floor -14 3)
it: -5
    1
user > (math:floor -14 -3)
it: 4
    -2
