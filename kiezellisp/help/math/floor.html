Name

math:floor

Usage

builtin-function

Syntax

(math:floor number)

Description

Returns the nearest integer in the direction of negative infinity.

user > (math:floor 6.99)
it: 6
user > (math:floor 14/3)
it: 4
user > (math:floor -14.3)
it: -15
