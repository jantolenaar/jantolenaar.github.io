Name

math:round

Usage

builtin-function

Syntax

(math:round number &optional (denominator 1))

Description

Returns a multiple value with the quotient and remainder of the division. The quotient is rounded to the nearest integer and to the nearest even integer when the nearest integers are exactly half an integer away.

user > (math:truncate 6)
it: 6
    0
user > (math:truncate 14 3)
it: 4
    2
user > (math:truncate 14/2 3/2)
it: 4
    1
user > (math:truncate 14.0 3.0)
it: 4
    2.0
user > (math:truncate 14 -3)
it: -4
    2
user > (math:truncate -14 3)
it: -4
    -2
user > (math:truncate -14 -3)
it: 4
    -2
