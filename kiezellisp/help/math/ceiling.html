Name

math:ceiling

Usage

builtin-function

Syntax

(math:ceiling number &optional (denominator 1))

Description

Returns a multiple value with the quotient and remainder of the division. The quotient is rounded to the nearest integer in the direction of positive infinity.

user > (math:ceiling 6)
it: 6
    0
user > (math:ceiling 14 3)
it: 5
    -1
user > (math:ceiling 14/2 3/2)
it: 14/3
    0
user > (math:ceiling 14.0 3.0)
it: 5
    -1.0
user > (math:ceiling 14 -3)
it: -4
    2
user > (math:ceiling -14 3)
it: -4
    -2
user > (math:ceiling -14 -3)
it: 5
    1
