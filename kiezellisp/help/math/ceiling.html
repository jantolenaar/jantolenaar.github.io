Name

math:ceiling

Usage

builtin-function

Syntax

(math:ceiling number)

Description

Returns the nearest integer in the direction of positive infinity.

make-html > (math:ceiling 6.99)
it: 7
make-html > (math:ceiling 14/3)
it: 5
make-html > (math:ceiling -14.3)
it: -14
