Name

math:ceiling

Usage

builtin-function

Syntax

(math:ceiling number)

Description

Returns the nearest integer in the direction of positive infinity.

user > (math:ceiling 6.99)
it: 7
user > (math:ceiling 14/3)
it: 5
user > (math:ceiling -14.3)
it: -14
