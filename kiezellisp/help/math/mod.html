Name

math:mod

Usage

builtin-function

Syntax

(math:mod number1 number2)

Description

Returns the remainder of the division of two numbers, i. e. the second value returned by math:floor.

user > (math:mod 7 3)
it: 1
user > (math:mod -7 3)
it: 2
user > (math:mod 7 -3)
it: -2
user > (math:mod -7 -3)
it: -1

See also

math:floor math:rem
