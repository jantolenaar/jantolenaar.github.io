Name

finish

Usage

loop clause

Syntax

(finish)

Description

When used within the main body of a loop, transfers control to the finally clause of that loop.

user > (loop
           (for i in '(1 2 3 a 5 6))
           (when (symbol? i)
               (finish))
           (collecting i)
           (finally (collecting 999)))
it: (1 2 3 999)
