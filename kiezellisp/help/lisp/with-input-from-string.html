Name

with-input-from-string

Usage

macro

Syntax

(with-input-from-string (sym text) &body forms)

Description

Binds sym to a lisp reader on the string text and executes forms as a do block.

user > (with-input-from-string ($stdin "aap noot mies") (read) (read-all))
it: (noot mies)
