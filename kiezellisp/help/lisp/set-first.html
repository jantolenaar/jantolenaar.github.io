Name

set-first

Usage

builtin-function

Syntax

(set-first seq item)

Description

Sets the first element of seq to item.

make-html > (let seq '(a 2 3 4))
it: seq
make-html > (set-first seq 'a)
it: a
make-html > (macroexpand '(setf (first seq) 'a))
it: (set-first seq 'a)
