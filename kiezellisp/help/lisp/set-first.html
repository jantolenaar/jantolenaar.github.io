Name

set-first

Usage

builtin-function

Syntax

(set-first seq item)

Description

Sets the first element of seq to item.

user > (let seq '(a 2 3 4))
it: seq
user > (set-first seq 'a)
it: a
user > (macroexpand '(setf (first seq) 'a))
it: (set-first seq 'a)
