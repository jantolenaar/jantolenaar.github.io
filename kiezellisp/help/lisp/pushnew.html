Name

pushnew

Usage

macro

Syntax

(pushnew item place)

Description

Prepends item to the list place if item is not in in the list and updates the setfable place to the new list.

user > (let a '(1 2 3))
it: a
user > (pushnew 4 a)
it: (4 1 2 3)
user > (pushnew 3 a)
it: (4 1 2 3)
