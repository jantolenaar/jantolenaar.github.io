Name

count

Usage

builtin-function

Syntax

(count predicate seq)

Description

Counts the number of elements in seq that match the predicate function.

make-html > (count odd? '(1 2 3 4 5))
it: 3
make-html > (count #(odd? (.length %1)) '("aap" "noot" "mies"))
it: 1
