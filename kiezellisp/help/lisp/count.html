Name

count

Usage

builtin-function

Syntax

(count predicate seq)

Description

Counts the number of elements in seq that match the predicate function.

user > (count odd? '(1 2 3 4 5))
it: 3
user > (count #(odd? (.length %1)) '("aap" "noot" "mies"))
it: 1
