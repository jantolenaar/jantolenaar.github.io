Name

interpose

Usage

builtin-function

Syntax

(interpose separator seq)
(interpose separator)

Description

Returns a new list with separator inserted between the items of seq. The second form returns a transducer.

user > (interpose '- '(a b c d))
it: (a - b - c - d)
