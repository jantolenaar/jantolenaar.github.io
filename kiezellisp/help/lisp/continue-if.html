Name

continue-if

Usage

loop clause

Syntax

(continue-if test)

Description

Starts the next iteration of the current loop, while or foreach loop if test condition is true.

user > (loop
           (for i in (series 5))
           (print-line i)
           (continue-if
               (odd? i))
           (print-line i))
