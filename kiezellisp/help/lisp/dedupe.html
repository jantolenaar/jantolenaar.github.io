Name

dedupe

Usage

builtin-function

Syntax

(dedupe seq)
(dedupe test seq)
(dedupe)
(dedupe test)

Description

Returns a lazy sequence without consecutive duplicates. The third and fourth forms return a transducer. The default equality test is =.

make-html > (dedupe '(1 2 2 3 3 4 5))
it: (1 2 3 4 5)
