Name

drop-while

Usage

builtin-function

Syntax

(drop-while predicate seq)
(drop-while predicate)

Description

Skips elements of seq while they satisfy the predicate. The second form returns a transducer.

make-html > (drop-while odd? '(1 1 2 3 4 5))
it: (2 3 4 5)
