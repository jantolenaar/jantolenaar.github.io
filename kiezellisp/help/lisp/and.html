Name

and

Usage

special-form/function

Syntax

(and expr*)

Description

Returns the value of the first argument that evaluates as false. Otherwise returns the value of the last argument. As a special form and evaluates as few arguments as possible.

user > (let n 12)
it: n
user > (and (number? n)
            (< n 13))
it: true
