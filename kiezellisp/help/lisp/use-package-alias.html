Name

use-package-alias

Usage

builtin-function

Syntax

(use-package-alias package-name nick-name)

Description

Gives the package with the long offical name package-name a short nick-name.

make-html > (package "very-annoying-name")
it: <Package Name="very-annoying-name">
very-annoying-name > (use-package-alias "very-annoying-name" "van")
it: "van"
