Name

drop

Usage

builtin-function

Syntax

(drop count seq)
(drop count)

Description

Skips count elements of seq and returns the remainder, which may be empty. The second form returns a transducer.

make-html > (drop 3 '(a b c d e f))
it: (d e f)
make-html > (drop 33 '(a b c d e f))
it: null
