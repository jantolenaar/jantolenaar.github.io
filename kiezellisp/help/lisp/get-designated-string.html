Name

get-designated-string

Usage

builtin-function

Syntax

(get-designated-string target)

Description

Returns the string representation of a keyword name, a symbol name or a value type.

user > (get-designated-string 'aap)
it: "aap"
user > (get-designated-string :aap)
it: "aap"
user > (get-designated-string "aap")
it: "aap"
user > (get-designated-string 123)
123 is not a symbol or keyword
