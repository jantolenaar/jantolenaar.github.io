Name

filter

Usage

builtin-function

Syntax

(filter predicate seq)
(filter predicate)

Description

Returns a lazy list containing the elements of seq that satisfy the function predicate. The second form returns a transducer.

user > (filter odd? (series 1 10))
it: (1 3 5 7 9)
