Name

bit-shift-right

Usage

builtin-function

Syntax

(bit-shift-right number1 number2)

Description

Returns the bitwise right algebraic shift of number1 by number2 bits.

user > (bit-shift-right 3 1)
it: 1
user > (bit-shift-right -1 1)
it: -1
user > (bit-shift-right -3 1)
it: -2
