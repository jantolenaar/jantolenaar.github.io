Name

literal?

Usage

builtin-function

Syntax

(literal? expr)

Description

Returns true if expr is null, a value type, a number or a string.

make-html > (literal? 123)
it: true
make-html > (literal? "abc")
it: true
make-html > (literal? 'aap)
it: false
