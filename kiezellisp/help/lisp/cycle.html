Name

cycle

Usage

builtin-function

Syntax

(cycle seq)

Description

Returns a lazy sequence of the elements of seq followed by the elements of seq and so on. Returns null if seq is empty.

user > (take 8 (cycle '(1 2 3)))
it: (1 2 3 1 2 3 1 2)
