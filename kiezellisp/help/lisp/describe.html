Name

describe

Usage

builtin-function

Syntax

(describe obj &optional show-non-public-members)

Description

Shows some information about obj. Called by the REPL command :d.

make-html > (describe "abc")
#s( :members #s( :length 3 )
    :type "System.String"
    :value "abc" )

make-html > (describe "abc" true)
#s( :members #s( :length 3 )
    :type "System.String"
    :value "abc" )
