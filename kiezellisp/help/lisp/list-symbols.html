Name

list-symbols

Usage

builtin-function

Syntax

(list-symbols package-name)

Description

Returns a list with the internal and external symbols of package package-name.

make-html > (list-symbols "shell")
it: (shell:$home
      shell:$local-application-data
      shell:$my-documents
