Name

missing-value?

Usage

builtin-function

Syntax

(missing-value? value)

Description

Returns true if value equals missing-value.

user > (defmacro some-return (&optional (a missing-value))
           (if (missing-value? a)
                 `(return)
               `(return ,a)))
it: some-return
user > (macroexpand '(some-return))
it: (return null)
user > (macroexpand '(some-return null))
it: (return null)
