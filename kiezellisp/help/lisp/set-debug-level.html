Name

set-debug-level

Usage

builtin-function

Syntax

(set-debug-level number)

Description

Debug level 0 provides no extra diagnostic information after an exception. Debug level 1 provides a lisp function call stack. Debug level 2 provides lexical and dynamic variable stack. $debugging is set to false for levels 0/1 and true for level 2.
