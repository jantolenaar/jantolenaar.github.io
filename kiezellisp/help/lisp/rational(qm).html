Name

rational?

Usage

builtin-function

Syntax

(rational? expr)

Description

Returns true if expr is a rational number.

user > (rational? (/ 1 2))
it: true
user > (rational? 1)
it: true
user > (rational? (math:pow 2 0.5))
it: false
