Name

rational?

Usage

builtin-function

Syntax

(rational? expr)

Description

Returns true if expr is a rational number.

make-html > (rational? (/ 1 2))
it: true
make-html > (rational? 1)
it: true
make-html > (rational? (math:pow 2 0.5))
it: false
