Name

not-every?

Usage

builtin-function

Syntax

(not-every? predicate seq)

Description

Returns true if at least one element of seq fails the function predicate.

user > (not-every? odd? '(1 2 3 4))
it: true
user > (not-every? odd? '(2 4))
it: true
user > (not-every? odd? '(1 3))
it: false
