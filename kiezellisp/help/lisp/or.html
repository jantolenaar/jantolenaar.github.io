Name

or

Usage

special-form/function

Syntax

(or expr*)

Description

Returns the value of the first argument that evaluates as true. Otherwise returns the value of the last argument. As a special form or evaluates as few arguments as possible.

user > (or ""
           "default")
it: "default"
user > (or "abc"
           "default")
it: "abc"
