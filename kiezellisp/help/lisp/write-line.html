Name

write-line

Usage

builtin-function

Syntax

(write-line item &key stream escape width padding pretty left right force base)

Description

Same as write but also writes a line feed.

user > (write-line 123 :width 8 :padding 0)
00000123
it: null
user > (write-line 123 :base 7 :escape false)
234
it: null
user > (write-line 123 :base 7 :escape true)
#7r234
it: null
user > (write-line "123" :escape false)
123
it: null
user > (write-line "123" :escape true)
"123"
it: null
