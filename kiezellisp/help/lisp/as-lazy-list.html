Name

as-lazy-list

Usage

builtin-function

Syntax

(as-lazy-list seq)

Description

Converts a sequence to a lazy list.

user > (let z (as-lazy-list (series 5)))
it: z
user > (first z)
it: 1
user > (rest z)
it: (2 3 4 5)
user > z
it: (1 2 3 4 5)
user > (last z)
it: (5)
user > z
it: (1 2 3 4 5)
