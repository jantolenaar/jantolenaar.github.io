Name

set-car

Usage

builtin-function

Syntax

(set-car list item)

Description

Assigns item to the first item of list.

user > (let z '(111 2 3))
it: z
user > (set-car z 111)
it: 111
user > z
it: (111 2 3)
user > (macroexpand '(setf (car z) 111))
it: (set-car z 111)
