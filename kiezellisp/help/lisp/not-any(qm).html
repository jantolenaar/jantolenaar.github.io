Name

not-any?

Usage

builtin-function

Syntax

(not-any? predicate seq)

Description

Returns true if no elements of seq satisfy the function predicate.

user > (not-any? odd? '(1 2 3 4))
it: false
user > (not-any? odd? '(2 4))
it: true
