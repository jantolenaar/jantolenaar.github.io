Name

make-package

Usage

builtin-function

Syntax

(make-package package-name)

Description

Returns an existing package or creates a new one. Called by the macro defpackage. The "lisp" package is automatically added to the use-list of the new package.

make-html > (make-package "csv")
it: <Package Name="csv">
