Name

last

Usage

builtin-function

Syntax

(last seq)
(last count seq)

Description

Returns the last 1 or count cons cells of a list.

make-html > (last '(1 2 3))
it: (3)
make-html > (last null)
it: null
make-html > (last 2 '(1 2 3))
it: (2 3)
make-html > (last 5 '(1 2 3))
it: (1 2 3)
