Name

last

Usage

builtin-function

Syntax

(last seq)
(last count seq)

Description

Returns the last 1 or count cons cells of a list.

user > (last '(1 2 3))
it: (3)
user > (last null)
it: null
user > (last 2 '(1 2 3))
it: (2 3)
user > (last 5 '(1 2 3))
it: (1 2 3)
