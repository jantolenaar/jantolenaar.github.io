Name

set-elt

Usage

special-form/function

Syntax

(set-elt obj index... value)

Description

Assigns value to the place index... of obj. A multi-dimensional array uses more than one index.

make-html > (let obj (new null))
it: obj
make-html > (set-elt obj "name" "mies")
it: "mies"
make-html > (elt obj "name")
it: "mies"
make-html > (attr obj 'name)
it: "mies"
make-html > (macroexpand '(setf (elt obj "name") "mies"))
it: (set-elt obj "name" "mies")
