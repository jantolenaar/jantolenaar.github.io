Name

zip

Usage

builtin-function

Syntax

(zip &rest seqs)

Description

Returns a lazy sequence consisting a vector with all first elements, a vector with all second elements, and so on.

make-html > (zip '(1 2 3) '(4 5 6 7))
it: (#v(1 4) #v(2 5) #v(3 6))
make-html > (zip '(1 2 3) (repeat '-))
it: (#v(1 -) #v(2 -) #v(3 -))
