Name

index-of

Usage

builtin-function

Syntax

(index-of item seq)
(index-of item test seq)

Description

Returns the position of item in seq. The default equality test is =.

user > (index-of 3 '(1 2 3 4))
it: 2
user > (index-of 3 '(1 2 4))
it: null
