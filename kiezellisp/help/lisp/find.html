Name

find

Usage

builtin-function

Syntax

(find predicate seq)

Description

Returns a list with the value and position of the first item in seq that satisfies the function predicate. Returns null if not found.

make-html > (find odd? '(2 3 4))
it: (3 1)
make-html > (find odd? '(2 4))
it: null
