Name

list-internal-symbols

Usage

builtin-function

Syntax

(list-internal-symbols package-name)

Description

Returns a list with the non-exported symbols of package package-name.

user > (list-internal-symbols "shell")
it: (shell::args
      shell::destination
      shell::forms
      shell::head
      shell::old-path
      shell::parts
      shell::replace-head
      shell::semicolon
      shell::source
      shell::unix
      shell::windows)
