Name

open-log

Usage

builtin-function

Syntax

(open-log path)

Description

Returns a TextWriter to write safely to the file path-yyyy-MM-dd.txt. Each call to the lisp functions print, print-line,writeandwrite-line` results in one write to the log file.

(setf $stdlog (open-log "/temp/my-log"))
(print-line $stdlog "hello, world")
