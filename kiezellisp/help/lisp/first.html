Name

first

Usage

builtin-function

Syntax

(first seq)
(first predicate seq)

Description

Returns the first (matching) element in a seq; or null if there is no first element.

user > (first '(1 2 3 4 5 6))
it: 1
user > (first even? '(1 2 3 4 5 6))
it: 2
