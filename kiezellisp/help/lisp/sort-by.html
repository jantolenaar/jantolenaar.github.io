Name

sort-by

Usage

builtin-function

Syntax

(sort-by key seq)
(sort-by key comparer seq)

Description

Sorts seq. The function key procedure the values to compare. The default comparer is the inbuilt function compare.

user > (sort-by length '("hello" "goodbye" "doei" "farewell"))
it: ("doei" "hello" "goodbye" "farewell")
