Name

macroexpand

Usage

builtin-function

Syntax

(macroexpand expr &optional environment)

Description

Returns the macro expansion of expr by repeatedly calling macroexpand-1.

make-html > (macroexpand '(setf (car x) 123))
it: (set-car x 123)
make-html > (macroexpand '(while a b))
it: (block LOOP
      (block TESTS
        (block MAIN
          (if (not a)
            (leave LOOP))
          b)
        (redo TESTS)))
