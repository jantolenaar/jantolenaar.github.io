Name

nth

Usage

builtin-function

Syntax

(nth pos seq)

Description

Returns the element at position pos in seq, or null if there is no element at that position.

make-html > (nth 0 '(a b c))
it: a
make-html > (nth 2 '(a b c))
it: c
make-html > (nth 9 '(a b c))
it: null
make-html > (nth -1 '(a b c))
it: null
