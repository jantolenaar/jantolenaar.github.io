Name

nth

Usage

builtin-function

Syntax

(nth pos seq)

Description

Returns the element at position pos in seq, or null if there is no element at that position.

user > (nth 0 '(a b c))
it: a
user > (nth 2 '(a b c))
it: c
user > (nth 9 '(a b c))
it: null
user > (nth -1 '(a b c))
it: null
