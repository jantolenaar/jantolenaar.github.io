Name

reductions

Usage

builtin-function

Syntax

(reductions reducer seq)
(reductions reducer seed seq)

Description

Returns a lazy list of every intermediate value produced by the function reduce called with the same arguments.

make-html > (reductions + (series 10))
it: (1 3 6 10 15 21 28 36 45 55)
make-html > (reductions + 100 (series 10))
it: (101 103 106 110 115 121 128 136 145 155)
