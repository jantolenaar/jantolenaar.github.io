Name

average

Usage

builtin-function

Syntax

(average seq)

Description

Computes the numeric average of a sequence of numbers.

user > (average '(1 2 3 4))
it: 5/2
user > (average (map .length '("aap" "noot" "mies")))
it: 11/3
