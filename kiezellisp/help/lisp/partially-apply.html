Name

partially-apply

Usage

function

Syntax

(partially-apply func &rest head-args)

Description

Creates a new function by partially applying arguments to the given function.

user > (def f (partially-apply string "de "))
it: f
user > (map f '(aap noot hond))
it: ("de aap" "de noot" "de hond")
