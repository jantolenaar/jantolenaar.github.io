Name

eql

Usage

builtin-function

Syntax

(eql a b)

Description

Returns true if a and b are equal objects (object.Equals).

user > (eql 'foo 'foo)
it: true
user > (eql "foo" "foo")
it: true
user > (eql 1 1)
it: true
user > (eql 1 1.0)
it: false
