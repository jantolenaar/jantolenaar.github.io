Name

eql

Usage

builtin-function

Syntax

(eql a b)

Description

Returns true if a and b are equal objects (object.Equals).

make-html > (eql 'foo 'foo)
it: true
make-html > (eql "foo" "foo")
it: true
make-html > (eql 1 1)
it: true
make-html > (eql 1 1.0)
it: false
