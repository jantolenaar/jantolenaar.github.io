Name

union

Usage

builtin-function

Syntax

(union seq1 seq2)
(union test seq1 seq2)

Description

Returns the union of seq1 and seq2. The default equality test is =.

make-html > (union '(1 2 3) '(3 4 5))
it: (1 2 3 4 5)
