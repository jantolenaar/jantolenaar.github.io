Name

set-symbol-value

Usage

builtin-function

Syntax

(set-symbol-value sym value)

Description

Assigns value to the global variable sym.

user > (def z 0)
it: z
user > (set-symbol-value 'z 1)
it: 1
user > (setq z 2)
it: 2
user > (macroexpand '(setf (symbol-value 'z) 1))
it: (set-symbol-value 'z 1)
