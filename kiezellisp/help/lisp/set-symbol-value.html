Name

set-symbol-value

Usage

builtin-function

Syntax

(set-symbol-value sym value)

Description

Assigns value to the global variable sym.

make-html > (def z 0)
it: z
make-html > (set-symbol-value 'z 1)
it: 1
make-html > (setq z 2)
it: 2
make-html > (macroexpand '(setf (symbol-value 'z) 1))
it: (set-symbol-value 'z 1)
