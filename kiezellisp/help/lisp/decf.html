Name

decf

Usage

macro

Syntax

(decf place &optional (delta 1))

Description

Decrements the variable or any setf-able expression by delta.

make-html > (let a 3)
it: a
make-html > (decf a)
it: 2
make-html > a
it: 2
make-html > (def b '(1 1 3))
it: b
make-html > (decf (second b))
it: 0
make-html > b
it: (1 0 3)
