Name

decf

Usage

macro

Syntax

(decf place &optional (delta 1))

Description

Decrements the variable or any setf-able expression by delta.

user > (let a 3)
it: a
user > (decf a)
it: 2
user > a
it: 2
user > (def b '(1 1 3))
it: b
user > (decf (second b))
it: 0
user > b
it: (1 0 3)
