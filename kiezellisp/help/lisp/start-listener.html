Name

start-listener

Usage

builtin-function

Syntax

(start-listener &optional (port $repl-listener-port))

Description

Starts a secondary thread listening to the given TCP port number. A text editor may send snippets of Kiezellisp code to this port to be executed. This is feature is available in the kiezellisp-sublimetext download.
