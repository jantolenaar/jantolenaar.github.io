Name

parallel-map

Usage

builtin-function

Syntax

(parallel-map func seq)

Description

Returns a lazy sequence where each element of seq is transformed by the function func. Uses the Task Parallel Library.

user > (parallel-map length (vector "aap" "noot" "mies"))
it: (3 4 4)
user > (parallel-map #(odd? (length %1)) (vector "aap" "noot" "mies"))
it: (true false false)
