Name

assoc

Usage

builtin-function

Syntax

(assoc item alist &key key test)

Description

Returns the first Cons item in association list alist with a car matching item.

user > (assoc 'a '((b c) (a d) (e f)))
it: (a d)
user > (assoc 3 '(("bbb" c) ("a" d) ("e" f)) :key length)
it: ("bbb" c)
