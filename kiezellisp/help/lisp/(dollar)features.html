Name

$features

Usage

special-variable

Description

A list of keywords showing some properties of Kiezellisp running on this computer.

user > $features
it: (:ansi-terminal :kiezellisp
                    :kiezellisp-con
                    :linux
                    :netcore
                    :repl
                    :unix
                    :x64)
