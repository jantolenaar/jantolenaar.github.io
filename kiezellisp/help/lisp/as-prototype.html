Name

as-prototype

Usage

builtin-function

Syntax

(as-prototype obj)

Description

Adds the contents of a hashtable, the values of static properties of a type or the instance properties of an object to a prototype hashtable.

user > (as-prototype 3/4)
it: #s( :denominator 4
        :numerator 3
        :sign 1 )
user > :d
#s( :type "Kiezel.Prototype"
    :type-specifier null
    :value #s( :denominator 4
               :numerator 3
               :sign 1 ) )

user > (as-prototype #/a3/i)
it: #s( :match-timeout -00:00:00.0010000
        :options IgnoreCase
        :right-to-left false )
user > :d
#s( :type "Kiezel.Prototype"
    :type-specifier null
    :value #s( :match-timeout -00:00:00.0010000
               :options IgnoreCase
               :right-to-left false ) )
