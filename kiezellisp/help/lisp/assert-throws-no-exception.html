Name

assert-throws-no-exception

Usage

macro

Syntax

(assert-throws-no-exception expr)

Description

Throws an AssertFailedException if expr throws an exception. Generates no code when $debugging is false.

user > (assert-throws-no-exception (= xxxxxxxx 1))
Failed throws-no-exception: (= xxxxxxxx 1)
