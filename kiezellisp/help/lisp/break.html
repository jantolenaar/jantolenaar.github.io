Name

break

Usage

loop clause

Syntax

(break &optional val)

Description

Breaks out of the current loop, while or foreach loop skipping the finally clause and returning the value of val instead of any accumulated values.

user > (loop
           (break null))
it: null
user > (loop
           (break 123))
it: 123
