Name

load

Usage

builtin-function

Syntax

(load file &key (verbose $load-verbose) (print $load-print))

Description

Loads, parses and evaluates the contents of file file. If verbose is true, the name and true location of file are printed to standard output. If print is true, the result of every top-level statement is printed to standard output.
