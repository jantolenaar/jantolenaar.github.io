Name

print-line

Usage

builtin-function

Syntax

(print-line &rest items)
(print-line text-writer &rest items)

Description

As print but followed by a newline.

user > (print-line "Hello" ", " "World")
Hello, World

user > (write-line "Hello" :escape false)
Hello

user > (write-line "Hello" :escape true)
"Hello"
