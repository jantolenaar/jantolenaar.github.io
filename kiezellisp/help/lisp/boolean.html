Name

boolean

Usage

builtin-function

Syntax

(boolean a)

Description

Returns true if a is true, otherwise false.

make-html > (boolean 0)
it: true
make-html > (boolean 1)
it: true
make-html > (boolean null)
it: false
make-html > (boolean "")
it: false
make-html > (boolean "aaaa")
it: true
make-html > (boolean (vector))
it: false
make-html > (boolean (vector 1 2 3))
it: true
