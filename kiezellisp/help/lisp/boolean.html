Name

boolean

Usage

builtin-function

Syntax

(boolean a)

Description

Returns true if a is true, otherwise false.

user > (boolean 0)
it: true
user > (boolean 1)
it: true
user > (boolean null)
it: false
user > (boolean "")
it: false
user > (boolean "aaaa")
it: true
user > (boolean (vector))
it: false
user > (boolean (vector 1 2 3))
it: true
