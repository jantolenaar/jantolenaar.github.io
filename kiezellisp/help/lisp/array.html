Name

array

Usage

builtin-function

Syntax

(array &rest items)

Description

Creates a fixed length array of type object[] with the given contents.

user > (array 1 2 3)
it: [1 2 3]
user > :d
#s( :members #s( :is-fixed-size true
                 :is-read-only false
                 :is-synchronized false
                 :length 3
                 :long-length 3
                 :rank 1
                 :sync-root [1 2 3] )
    :type "System.Object[]"
    :value [1 2 3] )
