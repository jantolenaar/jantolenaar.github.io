Name

distinct

Usage

builtin-function

Syntax

(distinct seq)
(distinct test seq)
(distinct)
(distinct test)

Description

Returns a lazy sequence of distinct elements in seq. The third and fourth forms return a transducer. The default equality test is =.

user > (distinct '(1 2 2 3 3 4 5))
it: (1 2 3 4 5)
