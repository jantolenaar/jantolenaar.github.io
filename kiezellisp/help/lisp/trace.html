Name

trace

Usage

macro

Syntax

(trace expr)

Description

Conditionally writes the source code of expr and its value to $stdlog, depending on the value of $tracing. expr must not have side effects. Trace code is compiled away when running in production mode.

user > (do
           (let $tracing true)
           (trace (+ 1 2)))
(+ 1 2) => 3
it: null
