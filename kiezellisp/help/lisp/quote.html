Name

quote

Usage

special-form

Syntax

(quote expr)

Description

Returns expr as literal data instead of evaluation expr as code. Usually written as 'expr instead of (quote expr).

user > (quote a)
it: a
user > 'a
it: a
user > (quote (+ 3 4))
it: (+ 3 4)
user > '(+ 3 4)
it: (+ 3 4)
