Name

quote

Usage

special-form

Syntax

(quote expr)

Description

Returns expr as literal data instead of evaluation expr as code. Usually written as 'expr instead of (quote expr).

make-html > (quote a)
it: a
make-html > 'a
it: a
make-html > (quote (+ 3 4))
it: (+ 3 4)
make-html > '(+ 3 4)
it: (+ 3 4)
