Name

in-package

Usage

builtin-function

Syntax

(package name)

Description

Sets the current package. The special variable $package holds the value of the current package.

make-html > (package "test-not-found")
it: <Package Name="test-not-found">
test-not-found > (package "user")
it: <Package Name="user">

See also
