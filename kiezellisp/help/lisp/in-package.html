Name

in-package

Usage

builtin-function

Syntax

(in-package name)

Description

Sets the current package. The special variable $package holds the value of the current package.

user > (in-package "test-not-found")
Undefined package: test-not-found
user 1 > (in-package "user")
it: <Package Name="user">
