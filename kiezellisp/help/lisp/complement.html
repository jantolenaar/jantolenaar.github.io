Name

complement

Usage

builtin-function

Syntax

(complement func)

Description

Returns a function that applies not to the result of func.

make-html > (mapv odd? (series 5))
it: #v(true false true false true)
make-html > (mapv (complement odd?) (series 5))
it: #v(false true false true false)
