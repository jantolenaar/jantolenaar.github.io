Name

get-package

Usage

builtin-function

Syntax

(get-package name)

Description

Returns the package corresponding to name. Throws an exception if not found.

user > (get-package "bla-bla")
Undefined package: bla-bla
user 1 > (get-package :lisp)
it: <Package Name="lisp">
