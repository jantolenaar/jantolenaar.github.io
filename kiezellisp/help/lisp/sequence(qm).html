Name

sequence?

Usage

builtin-function

Syntax

(sequence? expr)

Description

Returns true if expr is a sequence, i.e. null or IEnumerable.

user > (sequence? null)
it: true
user > (sequence? "abc")
it: false
user > (sequence? '(a b c))
it: true
