Name

sequence?

Usage

builtin-function

Syntax

(sequence? expr)

Description

Returns true if expr is a sequence, i.e. null or IEnumerable.

make-html > (sequence? null)
it: true
make-html > (sequence? "abc")
it: false
make-html > (sequence? '(a b c))
it: true
