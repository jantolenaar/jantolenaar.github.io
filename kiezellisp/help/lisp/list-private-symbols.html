Name

list-private-symbols

Usage

builtin-function

Syntax

(list-private-symbols package-name)

Description

Returns a list with the private symbols of package package-name.

make-html > (list-private-symbols "shell")
it: (shell::args
      shell::destination
      shell::forms
      shell::head
      shell::old-path
      shell::parts
      shell::replace-head
      shell::semicolon
      shell::source
      shell::unix
      shell::windows)
