Name

pairlis

Usage

builtin-function

Syntax

(pairlis keys values)
(pairlis keys values alist)

Description

Creates an association list.

user > (pairlis '(a b c) '(1 2 3))
it: ((c 3) (b 2) (a 1))
user > (pairlis '(a b c) '(1 2 3) '((d 4)))
it: ((c 3) (b 2) (a 1) (d 4))
