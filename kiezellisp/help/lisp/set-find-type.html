Name

set-find-type

Usage

builtin-function

Syntax

(set-find-type name type)

Description

Gives the name name to the prototype or System.Type type. The symbol name may now be used by the function type? and by the special form defmethod.

user > (set-find-type 'a (new :name "mies"))
it: #s(a :name "mies" )

See also

defmethod deftype type?
