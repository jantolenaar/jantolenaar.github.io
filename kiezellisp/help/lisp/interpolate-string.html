Name

interpolate-string

Usage

builtin-function

Syntax

(interpolate-string str)

Description

Evaluates lisp code embedded in str and returns a new string. String literals in source code are automatically interpolated. And that is why this example is so difficult to type in.

make-html > (string (string (string (string "aap#{(+ "))) "3 4)}noot")
it: "aap#{(+ 3 4)}noot"
make-html > (interpolate-string it)
it: "aap#{(+ 3 4)}noot"
