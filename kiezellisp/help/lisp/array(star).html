Name

array*

Usage

builtin-function

Syntax

(array* &rest items)

Description

Creates a fixed length array of type object[] with the given contents. The last item must be a sequence. Works like list*.

user > (array* 1 2 '(3 4))
it: [1 2 3 4]
user > :d
#s( :members #s( :is-fixed-size true
                 :is-read-only false
                 :is-synchronized false
                 :length 4
                 :long-length 4
                 :rank 1
                 :sync-root [1 2 3 4] )
    :type "System.Object[]"
    :value [1 2 3 4] )
