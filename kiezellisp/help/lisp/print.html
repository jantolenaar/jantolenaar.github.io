Name

print

Usage

builtin-function

Syntax

(print &rest items)
(print text-writer &rest items)

Description

Writes each item unescaped to standard output or the given TextWriter. When printing a list or vector, the elements are separated by spaces.

make-html > (print "Hello" "," "World")
Hello,World
make-html > (print $stdout '(1 2 3))
1 2 3
make-html > (write "Hello" :escape false)
Hello
make-html > (write "Hello" :escape true)
"Hello"
