Name

print

Usage

builtin-function

Syntax

(print &rest items)
(print text-writer &rest items)

Description

Writes each item unescaped to standard output or the given TextWriter. When printing a list or vector, the elements are separated by spaces.

user > (print "Hello" "," "World")
Hello,Worldit: null
user > (print $stdout '(1 2 3))
1 2 3it: null
user > (write "Hello" :escape false)
Helloit: null
user > (write "Hello" :escape true)
"Hello"it: null
