Name

map-indexed

Usage

function

Syntax

(map-indexed func &rest seqs)
(map-indexed func)

Description

Equivalent to (apply map func (range) seqs). The second form returns a transducer.

user > (map-indexed vector '(a b c))
it: (#v(0 a) #v(1 b) #v(2 c))
