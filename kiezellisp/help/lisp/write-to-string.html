Name

write-to-string

Usage

builtin-function

Syntax

(write-to-string item &key escape width padding pretty left right force base)

Description

Returns a string written to a StringWriter stream by the function write.

user > (write-to-string 123 :width 6 :padding "0" :base 9)
it: "#9r146"
