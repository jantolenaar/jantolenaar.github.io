Name

throw-error

Usage

builtin-function

Syntax

(throw-error &rest args)

Description

Throws an exception of type LispException. The arguments are combined as if by the function string.

user > (let n 2)
it: n
user > (throw-error "There are " n " things wrong.")
There are 2 things wrong.
