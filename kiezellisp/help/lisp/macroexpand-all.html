Name

macroexpand-all

Usage

builtin-function

Syntax

(macroexpand-all form)
(macroexpand-all form env)

Recursively expands all macros in form.

user > (macroexpand-all '(do (var x 1) (while (test x) (incf x))))
it: (do
      (var x 1)
      (block LOOP
        (block TESTS
          (block MAIN
            (if (not (test x))
              (leave LOOP))
            (do
              (let temp::new-1 (inc x))
              (setq x temp::new-1)))
          (redo TESTS))))
