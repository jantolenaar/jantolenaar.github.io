Name

set

Usage

builtin-function

Syntax

(set sym value)

Description

Assigns value to the global or dynamic variable named by the value of sym. Does not work for lexical variables.

make-html > (undef 'a137)

make-html > (setq x 'a137)
it: a137
make-html > (set x 1)
Undefined variable: make-html::a137
make-html 1 > (def a137 2)
it: a137
make-html 1 > (set x 1)
it: 1
make-html 1 > a137
it: 1
