Name

set

Usage

builtin-function

Syntax

(set sym value)

Description

Assigns value to the global or dynamic variable named by the value of sym. Does not work for lexical variables.

user > (undef 'a137)

user > (setq x 'a137)
it: a137
user > (set x 1)
Undefined variable: user:a137
user 1 > (def a137 2)
it: a137
user 1 > (set x 1)
it: 1
user 1 > a137
it: 1
