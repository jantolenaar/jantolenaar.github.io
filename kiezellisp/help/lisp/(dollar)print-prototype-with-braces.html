Name

$print-prototype-with-braces

Usage

special-variable

Description

When true, prototypes are printed like {...} otherwise like #s(...).

user > (do
           (let $print-prototype-with-braces true)
           (write-line (new :a 1 :b 2)))
{ :a 1 :b 2 }
it: null
user > (do
           (let $print-prototype-with-braces false)
           (write-line (new :a 1 :b 2)))
#s(:a 1 :b 2)
it: null
