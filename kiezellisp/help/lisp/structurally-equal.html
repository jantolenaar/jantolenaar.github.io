Name

structurally-equal

Usage

builtin-function

Syntax

(structurally-equal obj1 obj2)

Description

Returns true if obj1 and obj2 are equal objects or sequences containing the same structurally_equal objects.

make-html > (structurally-equal 1 1)
it: true
make-html > (structurally-equal (series 5) (vector 1 2 3 4 5))
it: true
