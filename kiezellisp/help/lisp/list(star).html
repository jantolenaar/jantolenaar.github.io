Name

list*

Usage

builtin-function

Syntax

(list* &rest items)

Description

Returns a list containing the given items. The last item is a list that becomes the tail of the new list.

make-html > (list* 1 2 '(3 4))
it: (1 2 3 4)
make-html > (list 1 2 '(3 4))
it: (1 2 (3 4))
