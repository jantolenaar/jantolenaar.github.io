Name

multiple-setf

Usage

macro

Syntax

(multiple-setf (place*) value)

Description

Assigns places with the elements of a sequence or tuple.

user > (do
           (var a '(1 2))
           (var b null)
           (var c (new :city "leiden"))
           (multiple-setf ((car a) b (.city c)) '(one two three))
           (vector a b c))
it: #v((one 2) two #s(:city three))
