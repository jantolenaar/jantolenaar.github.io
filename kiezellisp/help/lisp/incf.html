Name

incf

Usage

macro

Syntax

(incf place &optional (delta 1))

Description

Increments the value of the setfable place by delta.

make-html > (let x 3)
it: x
make-html > (incf x)
it: 4
make-html > x
it: 4
make-html > (let a '(1 2 3))
it: a
make-html > (incf (second a))
it: 3
make-html > a
it: (1 3 3)
