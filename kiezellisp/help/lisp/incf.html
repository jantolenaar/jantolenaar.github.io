Name

incf

Usage

macro

Syntax

(incf place &optional (delta 1))

Description

Increments the value of the setfable place by delta.

user > (let x 3)
it: x
user > (incf x)
it: 4
user > x
it: 4
user > (let a '(1 2 3))
it: a
user > (incf (second a))
it: 3
user > a
it: (1 3 3)
