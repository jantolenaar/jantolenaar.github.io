Name

export-symbol

Usage

builtin-function

Syntax

(export-symbol name)
(export-symbols &rest names)

Description

Exports the symbol with the given name from the current package. This function does not create symbols, it merely marks the name as being exported. Called by the macro defpackage.

make-html > (export-symbol "export-symbol")
