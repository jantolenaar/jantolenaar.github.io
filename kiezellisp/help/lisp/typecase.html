Name

typecase

Usage

macro

Syntax

(typecase keyform normal-clause* [otherwise-clause])

Description

normal-clause := '(' type form* ')'
otherwise-clause := '(' {true|otherwise} form* ')'
type := symbol corresponding to a (imported) class
