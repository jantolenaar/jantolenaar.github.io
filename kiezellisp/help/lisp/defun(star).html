Name

defun*

Usage

special-form

Syntax

(defun* name (arg* [doc-string] form*) (. ".."))

Description

Defines a multi-arity function.

make-html > (defun* f
                (() (self 0))
                ((x) (self x x))
                ((x y) (+ x y)))
it: f
make-html > (f)
it: 0
make-html > (f 3)
it: 6
make-html > (f 3 4)
it: 7
