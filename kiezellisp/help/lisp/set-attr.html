Name

set-attr

Usage

special-form/function

Syntax

(set-attr object property value)

Description

Assigns value to property of object.

make-html > (let obj (new null))
it: obj
make-html > (set-attr obj 'name "mies")
it: "mies"
make-html > (attr obj 'name)
it: "mies"
make-html > (.name obj)
it: "mies"
make-html > (macroexpand '(setf (.name obj) "mies"))
it: (set-attr obj "name" "mies")
