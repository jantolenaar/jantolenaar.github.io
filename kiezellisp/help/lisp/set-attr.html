Name

set-attr

Usage

special-form/function

Syntax

(set-attr object property value)

Description

Assigns value to property of object.

user > (let obj (new null))
it: obj
user > (set-attr obj 'name "mies")
it: "mies"
user > (attr obj 'name)
it: "mies"
user > (.name obj)
it: "mies"
user > (macroexpand '(setf (.name obj) "mies"))
it: (set-attr obj "name" "mies")
