Name

set-cdr

Usage

builtin-function

Syntax

(set-cdr list item)

Description

Assigns the list item to the rest or cdr of list.

make-html > (let z '(1 111))
it: z
make-html > (set-cdr z '(111))
it: (111)
make-html > z
it: (1 111)
make-html > (macroexpand '(setf (cdr z) '(111)))
it: (set-cdr z '(111))
