Name

multiple-value-list

Usage

special-form

Syntax

(multiple-value-list form)

Description

Returns a list from the multiple values produced by evaluating form.

user > (multiple-value-list (values 1 2 3))
it: (1 2 3)
user > (multiple-value-list (values))
it: null
