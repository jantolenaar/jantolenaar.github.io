Name

remove

Usage

builtin-function

Syntax

(remove predicate seq)
(remove predicate)

Description

Returns a lazy list containing the elements of seq that do not satisfy the function predicate. The second form returns a transducer.

user > (remove odd? (series 10))
it: (2 4 6 8 10)
user > (filter odd? (series 10))
it: (1 3 5 7 9)
