Name

gentemp

Usage

builtin-function

Syntax

(gentemp prefix)
(gentemp)

Description

Returns a new symbol in the "temp" package. For use by macro writers.

user > (gentemp "xyz")
it: temp:xyz-1
user > (gentemp)
it: temp:temp-2
