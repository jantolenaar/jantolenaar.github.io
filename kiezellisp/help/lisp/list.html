Name

list

Usage

builtin-function

Syntax

(list &rest items)

Description

Returns a list containing the given items.

user > (list 1 2 3)
it: (1 2 3)
user > (list 'a 'b 'c)
it: (a b c)
user > '(a b c)
it: (a b c)
