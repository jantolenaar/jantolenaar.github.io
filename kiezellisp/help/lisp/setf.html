Name

setf

Usage

macro

Syntax

(setf place value)

Description

Assigns value to place. place may be transformed by (symbol) macro expansion. See the file core/setf.

make-html > (var a '(1 2 3))
it: a
make-html > (setf (second a) 'two)
it: two
make-html > a
it: (1 two 3)
make-html > (macroexpand '(setf (second a) 'two))
it: (set-second a 'two)
