Name

setf

Usage

macro

Syntax

(setf place value)

Description

Assigns value to place. place may be transformed by (symbol) macro expansion. See the file core/setf.

user > (var a '(1 2 3))
it: a
user > (setf (second a) 'two)
it: two
user > a
it: (1 two 3)
user > (macroexpand '(setf (second a) 'two))
it: (set-second a 'two)
