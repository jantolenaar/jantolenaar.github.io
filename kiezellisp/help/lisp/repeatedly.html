Name

repeatedly

Usage

builtin-function

Syntax

(repeatedly count func)
(repeatedly func)

Description

Returns a lazy sequence of the values returned by repeated calls to func. The second form yields an infinite sequence.

user > (repeatedly 10 (lambda null "a"))
it: ("a" "a" "a" "a" "a" "a" "a" "a" "a" "a")
