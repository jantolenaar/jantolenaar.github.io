Name

eval

Usage

builtin-function

Syntax

(eval expr env)
(eval expr)

Description

Evaluates a lisp expression.

user > (eval '(+ 3 4))
it: 7
user > (let n 3)
it: n
user > (+ 3 n)
it: 6
user > (eval '(+ 3 n))
it: 6
