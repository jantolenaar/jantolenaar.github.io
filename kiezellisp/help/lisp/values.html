Name

values

Usage

builtin-function

Syntax

(values &rest args)

Description

Returns a multiple value. The expression (values) may be used to signal the absence of a return value.

user > (values 1 2 3)
it: 1
    2
    3
user > (def z (values 1 2 3))
it: z
user > z
it: 1
