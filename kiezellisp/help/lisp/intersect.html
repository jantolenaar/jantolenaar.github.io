Name

intersect

Usage

builtin-function

Syntax

(intersect seq1 seq2)
(intersect test seq1 seq2)

Description

Returns the intersection of two sequences. The default equality test is =.

make-html > (intersect '(1 2 3 4) '(2 4 6))
it: (2 4)
