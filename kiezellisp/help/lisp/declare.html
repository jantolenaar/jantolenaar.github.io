Name

declare

Usage

special-form

Syntax

(declare &rest args)

Description

Gives compiling instructions to the compiler.

(declare (ignore a b c...))

An ignore declaration instructs the compiler to not emit warning messages about already defined lexical variables that are unreferenced.
