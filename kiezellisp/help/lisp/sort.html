Name

sort

Usage

builtin-function

Syntax

(sort seq)
(sort comparer seq)

Description

Sorts seq. The default comparer is the inbuilt function compare.

make-html > (sort "hello, world")
it: (#\space #\, #\d #\e #\h #\l #\l #\l #\o #\o #\r #\w)
make-html > (sort '("hello" "goodbye" "doei" "farewell"))
it: ("doei" "farewell" "goodbye" "hello")
