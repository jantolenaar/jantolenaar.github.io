Name

merge

Usage

builtin-function

Syntax

(merge seq1 seq2)
(merge comparer seq1 seq2)

Description

Merges two ordered sequences into one. Used by the function sort. The default comparer is compare.

user > (merge '(1 3 5) '(2 2 4 6))
it: (1 2 2 3 4 5 6)
