Name

assoc-if

Usage

builtin-function

Syntax

(assoc-if predicate alist &key key)

Description

Returns the first Cons item in association list alist with a car satisfying predicate.

user > (assoc-if odd? '((2 c) (4 d) (5 f)))
it: (5 f)
