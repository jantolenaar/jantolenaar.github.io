Name

zero?

Usage

builtin-function

Syntax

(zero? number)

Description

Returns true if the number number equals zero. Fails if number is not a number.

make-html > (zero? 0)
it: true
make-html > (zero? 1)
it: false
make-html > (zero? "aaa")
Not a number
