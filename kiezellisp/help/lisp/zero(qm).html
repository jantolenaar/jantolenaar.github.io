Name

zero?

Usage

builtin-function

Syntax

(zero? number)

Description

Returns true if the number number equals zero. Fails if number is not a number.

user > (zero? 0)
it: true
user > (zero? 1)
it: false
user > (zero? "aaa")
Not a number
