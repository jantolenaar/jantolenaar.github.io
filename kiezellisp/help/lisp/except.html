Name

except

Usage

builtin-function

Syntax

(except seq1 seq2)
(except test seq1 seq2)

Description

Returns all elements from seq1 that are not also in seq2. The default equality test is =.

make-html > (except '(1 2 3 4) '(3))
it: (1 2 4)
