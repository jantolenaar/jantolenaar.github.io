Name

read-from-string

Usage

builtin-function

Syntax

(read-from-string text &optional (eof-error? true) (eof-value null))

Description

Reads an expression from the string text.

user > (read-from-string "(+ 3 4)")
it: (+ 3 4)
user > (read-from-string "")
read-from-string: unexpected EOF
user 1 > (read-from-string "" false)
it: null
user 1 > (read-from-string "" false 42)
it: 42
