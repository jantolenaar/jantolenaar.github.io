Name

flatten

Usage

builtin-function

Syntax

(flatten seq &optional (depth int32:MAX-VALUE))

Description

Returns a lazy sequence with subsequences replaced by their elements up to a certain depth.

user > (flatten '(a (b c (d e) f)))
it: (a b c d e f)
user > (flatten '(a (b c (d e) f)) 1)
it: (a b c (d e) f)
