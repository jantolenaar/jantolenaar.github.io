Name

make-symbol

Usage

builtin-function

Syntax

(make-symbol &rest args)

Description

Returns an existing symbol or creates a new one. For use by macros.

user > (let a 'test1)
it: a
user > (let b (make-symbol "test" 1))
it: b
user > (eq a b)
it: true
