Name

make-symbol

Usage

builtin-function

Syntax

(make-symbol &rest args)

Description

Returns an existing symbol or creates a new one. For use by macros.

make-html > (let a 'test1)
it: a
make-html > (let b (make-symbol "test" 1))
it: b
make-html > (eq a b)
it: true
